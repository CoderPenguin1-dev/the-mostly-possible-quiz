# The Mostly Possible Quiz!
My CS Principles Group Project.

Meant to be a clone of The Impossible Quiz.

# Running
As with most other python applications, its simple.

Clone the repository.
```
git clone https://github.com/CoderPenguin1-dev/the-mostly-possible-quiz.git
```

Go into source directory.
```
cd the-mostly-possible-quiz/src
```

Run the game!
```
python main.py
```
